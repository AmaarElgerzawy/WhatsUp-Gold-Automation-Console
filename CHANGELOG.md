# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added

- Centralized credential management system with localStorage
- Custom naming for saved configs and logs
- Rename functionality for config and log files
- Download functionality for config and log files
- Improved log/config display with full-width wrapping
- Excel-like report schedule grid with dark theme
- Modern dark UI theme with glassmorphism effects
- Shared configuration modules for Python scripts (db_config, backup_config, report_config)

### Changed

- Enhanced UI/UX across all pages
- Improved file display to prevent horizontal scrolling
- Better error handling and validation
- Refactored backend API endpoints for better organization

### Fixed

- Resolved 422 errors when using saved credentials
- Fixed white background issue in report schedule grid
- Improved credential auto-selection behavior

## [0.1.0] - Initial Release

### Features

- Bulk device operations (add, update, delete)
- Router command execution (simple and interactive)
- Config backup browsing
- Report schedule management
- History and logs viewing
